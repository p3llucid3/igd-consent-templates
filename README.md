# IGD Informed Consent Templates

This repository contains **generic informed consent templates** for emergency
and hospital medical procedures.

## Purpose
- Used as **data source** for a Chrome Extension
- Provides standardized wording for informed consent
- Contains **NO patient data**
- Contains **NO hospital-specific SOP**

## Safety & Privacy
- Templates are generic and non-identifiable
- No personal, institutional, or operational data included

## Versioning
Templates are versioned using a semantic version number
inside `templates/igd-master.json`.
